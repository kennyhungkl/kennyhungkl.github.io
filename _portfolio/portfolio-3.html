Overview

A grounded QA chatbot that answers questions from a curated set of web pages (wiki-like knowledge). It combines symbolic search (BM25) and dense retrieval (FAISS embeddings) and then reranks with a cross-encoder to ensure the most relevant contexts are fed to the LLM. The app is built with Streamlit and LangChain 0.3 and is organized into modular components for clean maintenance and extensibility.

Key Features

  • Streamlit UI for fast iteration and shareable demos.
  • Hybrid retrieval: BM25 keyword search + FAISS vector search.
  • Cross-encoder reranking for high-precision context selection.
  • Grounded answers with citations to source pages.
  • Composable LangChain 0.3 modules for loaders, chunkers, retrievers, rerankers, and synthesizers.
